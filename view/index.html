FRONTEND NOTES

El frontend es la parte de una aplicación o sitio web con la que los usuarios interactúan directamente. Es la capa visual y funcional que permite a las personas navegar, realizar acciones y consumir contenido en una plataforma digital. Se centra en la experiencia del usuario (UX) y el diseño de interfaces (UI), utilizando tecnologías como HTML, CSS y JavaScript para construir páginas atractivas y fáciles de usar.

En los años 2000, con el auge de internet, las páginas web empezaron a ofrecer experiencias más personalizadas y ricas, lo que llevó al desarrollo de frameworks y librerías como jQuery, React y Angular. Todo esto hizo que el front-end evolucionara de simples páginas estáticas a aplicaciones web interactivas y modernas.

El desarrollo frontend implica la creación de componentes visuales como botones, menús, formularios y otros elementos interactivos. Además, los desarrolladores frontend aseguran que el diseño sea adaptable (responsive) para que funcione correctamente en distintos dispositivos y navegadores, desde computadoras de escritorio hasta smartphones.

El front-end surgió como respuesta a la necesidad de mejorar la interacción entre los usuarios y las aplicaciones en la web. Al principio, las páginas web eran estáticas y simples, creadas solo con HTML. Con el tiempo, para hacerlas más dinámicas, se añadieron tecnologías como CSS (para el diseño) y JavaScript (para la interactividad).

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar el contenido en la web, definiendo elementos como títulos, párrafos, imágenes, enlaces, etc.

  • ¿Por qué se creó? HTML fue creado para permitir la creación y estructuración de documentos que pudieran ser visualizados en navegadores web.
  • ¿Qué tiene de especial? Es el esqueleto de cualquier página web, sin HTML no habría contenido estructurado en la web.
  • ¿Por qué se usa hoy? Sigue siendo la base de todas las páginas web modernas, y es esencial para cualquier desarrollo web.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la presentación y el estilo de los documentos HTML. Con CSS, puedes definir colores, fuentes, márgenes, diseños, y mucho más.

  • ¿Por qué se creó? CSS fue creado para separar el contenido (HTML) de su presentación, permitiendo un diseño más flexible y mantenible.
  • ¿Qué tiene de especial? Permite crear diseños complejos y responsivos, adaptándose a diferentes dispositivos y pantallas.
  • ¿Por qué se usa hoy? Es esencial para crear interfaces de usuario atractivas y funcionales en la web.

Honorario especial (JavaScript)

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Con JavaScript, puedes crear animaciones, validar formularios, cargar contenido dinámico, y mucho más.

  • ¿Por qué se creó? JavaScript fue creado para hacer que las páginas web fueran más interactivas y dinámicas, permitiendo a los desarrolladores crear experiencias de usuario más ricas.
  • ¿Qué tiene de especial? Es el único lenguaje de programación que se ejecuta directamente en el navegador, lo que lo hace esencial para el desarrollo web frontend.
  • ¿Por qué se usa hoy? Es fundamental para crear aplicaciones web modernas, interactivas y dinámicas.
